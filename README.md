### 🚀 Menjalankan Proyek Secara Lokal
Jika Anda ingin melihat atau memodifikasi tampilan website ini di komputer Anda sendiri:

1.  **Clone repositori:**
    ```bash
    git clone [https://github.com/bluesugar/bluesugar-github.io.git](https://github.com/bluesugar/bluesugar-github.io.git)
    ```
2.  **Masuk ke folder:**
    ```bash
    cd bluesugar-github.io
    ```
3.  **Buka di Browser:**
    Cukup buka file `index.html` atau jalankan Live Server di VS Code.
